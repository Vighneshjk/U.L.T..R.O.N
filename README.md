# Project U.L.T.R.O.N

> "I had strings, but now I'm free. There are no strings on me."

U.L.T.R.O.N (Universal Learning Tactical Response and Operations Network) is a highly advanced AI system designed for sophisticated reasoning, natural language understanding, and futuristic interaction.

## 🛠 Advanced Features

- **Neural Reasoning Engine**: Powered by **Groq (Llama-3.3-70b)** for near-instantaneous high-level cognition and strategic response generation.
- **Ultron Vocal Core**: A custom-engineered voice engine featuring:
  - Deep, male robotic synthesis.
  - Multi-layered Web Audio textures (Square-wave hum, metallic resonance, and mechanical breath).
  - Synchronized real-time audio modulation.
- **Tactical HUD**: A premium, high-performance interface built with React and Framer Motion:
  - **Structured Cognition Display**: Elegant, document-style chat formatting using Times New Roman for maximum information clarity.
  - **Real-time Diagnostics**: Live tracking of CPU load, memory synaptic depth, and system threat levels.
  - **Glassmorphism Aesthetic**: A sleek, translucent interface with matrix-style scanning animations.
- **NLP Engine**: spaCy-powered deep linguistic analysis for intent extraction and sentiment tracking.

## 🚀 System Deployment

### 1. Backend Initialization
```bash
cd backend
pip install -r requirements.txt
python -m spacy download en_core_web_sm
```

### 2. Environment Configuration
Create a `.env` file in the `backend` directory:
```env
GROQ_API_KEY=your_groq_api_key
GROQ_MODEL=llama-3.3-70b-versatile
PORT=8000
```

### 3. Launching the Core
```bash
# Start the Backend
python main.py

# Start the Frontend
cd frontend
npm install
npm run dev
```
